# Number Guessing Game
## A Guess the Number (with Difficulty Levels) Game
This is a guessing game where the computer generates a random number and the user tries to guess it.

It makes use of random.randint() to generate a number between 1 and 100.

We ask the user to input guesses and give feedback: “Too High”, “Too Low”.

We limit the number of guesses to 5–10 depending on the level.

## Features
- Allow user to choose difficulty: Easy (10 tries), Medium (7), Hard (5).
- Count attempts and print remaining guesses.
- Game ends on correct guess or when attempts run out.
